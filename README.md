A simple and reusable **C#/.NET password generator SDK** for creating secure passwords with ease.

## Why use NewPasswordGenerator?

- Generate passwords quickly in your .NET projects
- Lightweight and easy to integrate
- Built for reuse as a NuGet package
- Great starting point for custom password rules

## Features

- Secure password generation
- Clean C# implementation
- NuGet-friendly package structure
- Works well in modern .NET workflows

## Installation

Install from NuGet:

```bash
 dotnet add package NewPasswordGenerator
```
